# Electronics Store Azure Data Engineering Project

## Project Overview

This project demonstrates an end-to-end Azure Data Engineering solution for electronics store sales analytics using:

- Azure Data Factory
- Azure Databricks
- PySpark
- Delta Lake
- ADLS Gen2
- On-Prem SQL Server


---

# Architecture

On-Prem SQL Server
        ↓
Azure Data Factory
        ↓
ADLS Gen2 Raw Layer
        ↓
Azure Databricks PySpark
        ↓
Delta Lake Bronze / Silver / Gold
        ↓
Azure SQL
